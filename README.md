# App para mi campaña Poké RPGARS
App web progresiva para llevar mi campaña *Aventura en la región de Sinnoh* (RPGARS), basada en el *corebook* del proyecto [Pokérole](https://www.pokeroleproject.com/) y el set de datos [Pokerole-Data](https://github.com/Pokerole-Software-Development/Pokerole-Data). He diseñado esta app para mi uso personal y solo está disponible en español.
